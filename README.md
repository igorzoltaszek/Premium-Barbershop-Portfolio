# 💈 Cut & Shave - Premium Barbershop Website

Nowoczesna, w pełni responsywna strona internetowa dla salonu barberskiego, zaprojektowana z myślą o najwyższej wydajności i świetnym UX. Projekt stworzony jako element portfolio.

🔗 **[Zobacz stronę na żywo (Live Demo)](#)** *https://premiumbarbershopportfolio.netlify.app/*

## 🛠 Technologie

Projekt został zbudowany przy użyciu nowoczesnego stacku technologicznego:
* **[Astro](https://astro.build/)** - Framework do generowania ultraszybkich statycznych stron (SSG).
* **[Tailwind CSS v4](https://tailwindcss.com/)** - Utility-first framework CSS do szybkiego i spójnego stylowania (z wykorzystaniem najnowszej integracji z Vite).
* **Vanilla JS** - Lekkie skrypty do obsługi menu mobilnego i interaktywnych popupów (zero ciężkich bibliotek).

## ✨ Główne funkcje

* **Dark Mode Premium UI:** Elegancki, mroczny design z miedzianymi/bursztynowymi akcentami, idealnie pasujący do branży barberskiej.
* **Wzorowa Wydajność:** Dzięki Astro strona ładuje się błyskawicznie, przesyłając do przeglądarki tylko czysty HTML i CSS.
* **SEO & Local Business Ready:** Wdrożone zaawansowane meta tagi, Open Graph, Twitter Cards oraz dane strukturalne JSON-LD (Schema.org dla typu `HairSalon`), co ułatwia pozycjonowanie w lokalnych wynikach wyszukiwania.
* **Mobile-First Design:** Pełna responsywność na wszystkich urządzeniach z dedykowanym menu mobilnym (hamburger menu).
* **Interaktywny Mockup Rezerwacji:** Zaimplementowany globalny komponent Popup dla przycisków rezerwacji (gotowy do podpięcia pod systemy typu Booksy czy Moment.pl).
